## How to Run the Project Locally

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Irina-Kulish/UsersVue


2. **Install Dependencies:**
   npm install


3. **Run the Project:**
npm run serve


4. **Open in Your Browser:**
Navigate to http://localhost:8080 in your web browser.
